## Quick Actions

### Change Theme Quickly
1. Press **F1** (or **Start**)
2. Click first or second button under "Visual Theme"
3. Press **ESC** (or **B**)

### Adjust UI Size Quickly
1. Press **F1** (or **Start**)
2. Click desired scale percentage
3. Press **ESC** (or **B**)

### Check Controller
1. Press **F1** (or **Start**)
2. Look at "Controller" section
3. Name and status shown

## Common Use Cases

### "Text is too small"
1. Open settings (F1)
2. Click "125%" or "150%"
3. Close settings (ESC)
4. UI elements are now larger

### "I'm colorblind"
1. Open settings (F1)
2. Click "Colorblind Mode"
3. Close settings (ESC)
4. Colors now friendly for red-green colorblindness

### "Is my controller working?"
1. Open settings (F1)
2. Check "Controller" section
3. Should show your gamepad name
4. Try pressing buttons - status should say "Active"

### "Everything is too big/small"
1. Open settings (F1)
2. Click "Auto"
3. Close settings (ESC)
4. Game automatically picks best size for your screen

## Troubleshooting

### "Settings button does nothing"
- Try keyboard shortcut: **F1**
- Try gamepad: **Start button**
- Check console for errors

### "Theme changed but colors look wrong"
- Close settings and reopen
- Theme should be fully applied
- Check "Current:" text to confirm

### "UI Scale doesn't seem to work"
- Font sizes don't scale (engine limitation)
- Button sizes DO scale - look carefully
- Try 150% to see clear difference

### "Controller not detected"
- Make sure gamepad is plugged in
- Press a button to activate it
- Check if drivers are installed (Windows)
- Try reconnecting the controller
